# Projetos 3

Projetos 3 - Contei <br/>

## Installation

### Clone

- Clone this repo to your local machine using:
```shell
$ git clone https://github.com/ArthurPaes/Projetos-3
```
### Setup
> Enter the folder:
```shell
$ cd Projetoss-3
```
> Execute `yarn` or `npm instal` to install all the dependencies

```shell
$ yarn 
$ npm install
```

> Start the project:
```shell
$ yarn start
```
